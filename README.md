# Химчистка — Автоматизированное рабочее место менеджера

Android-приложение для управления химчисткой с Material Design 3.

## Технологии

- **Kotlin** + MVVM
- **Jetpack**: ViewModel, Flow, Navigation Component, Room, DataStore, WorkManager
- **Material Design 3** (Material You) — светлая и тёмная тема
- **Hilt** — Dependency Injection
- **MPAndroidChart** — графики и аналитика
- **ViewBinding** — типобезопасный доступ к View

## Функционал

### 1. Авторизация
- Экран входа с валидацией
- Роли: Менеджер / Администратор
- Хранение сессии через DataStore
- Учётные данные по умолчанию:
  - `admin` / `admin123` (Администратор)
  - `manager` / `manager123` (Менеджер)

### 2. Dashboard
- Количество активных / завершённых заказов
- Выручка за день и месяц
- Быстрые кнопки: Новый заказ, Клиенты, Отчёты, Склад

### 3. Управление заказами
- Создание, редактирование, удаление
- Автоматический расчёт стоимости по прайс-листу
- Поиск по номеру заказа или имени клиента
- Фильтрация по статусу (Принят / В работе / Готов / Выдан)

### 4. Управление клиентами
- CRUD-операции
- Поиск по имени и телефону
- История заказов клиента

### 5. Склад
- Учёт расходных материалов
- Уведомление о низком остатке

### 6. Отчёты
- Выручка и количество заказов за период
- Графики: круговая и столбчатая диаграммы
- Экспорт в PDF

### 7. Уведомления
- Напоминание о готовых заказах
- Уведомление о просроченных заказах

### 8. Настройки
- Переключение темы (светлая / тёмная / системная)
- Редактирование прайс-листа
- Резервное копирование базы данных
- Выход из системы

## Запуск

1. Откройте проект в Android Studio (Hedgehog или новее)
2. Дождитесь синхронизации Gradle
3. Подключите устройство или запустите эмулятор (API 24+)
4. Нажмите Run (Shift+F10)
5. Войдите: `admin` / `admin123` или `manager` / `manager123`
